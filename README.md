## Intalation
* Install [nodejs](https://nodejs.org/en/)
* Clone this repo
    git clone https://github.com/awkz/ionic-radio-stream
* Module instalation
    npm install

## configuration
edit url stream in **src\providers\radio-player**
    this.url = "your stream url";
